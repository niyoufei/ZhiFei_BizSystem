# 青天评标系统 - Windows 安装与使用（公司内部版）

本文适用于在公司内部、每台 **Windows** 电脑上安装并正式使用本系统。目标：**操作最简单、自动化程度高、一键启动**。

> **说明**：本系统同时支持 **macOS** 与 **Windows**。Mac 用户可参考 [README](../README.md) 或 [试用指南](试用指南.md)，使用 `make run` 或 `bash scripts/start.sh` 启动；功能与数据格式一致。

---

## 一、安装（每台电脑仅需一次）

### 1. 安装 Python 3.10 或更高版本

1. 打开 https://www.python.org/downloads/
2. 下载 **Windows installer (64-bit)** 并运行
3. **务必勾选** “Add python.exe to PATH”
4. 点击 “Install Now”，完成安装

### 2. 获取系统压缩包并解压

- 将 `ZhiFei_BizSystem` 项目文件夹复制到本机（如 `D:\青天评标系统` 或桌面）
- 确保路径中**尽量不要有空格和特殊字符**，避免部分电脑出现异常

### 3. 安装依赖（首次运行前）

在项目文件夹内**按住 Shift 右键 → “在终端中打开”**（或打开 cmd 后 `cd` 到项目目录），执行：

```bat
python -m pip install -r requirements.txt
```

若提示权限不足，可尝试以**管理员身份**打开命令提示符再执行上述命令。

## 二、使用（日常）

### 一键启动

1. 双击运行 **`scripts\start.bat`**
2. 等待几秒，浏览器会**自动打开**系统页面；若未打开，请手动访问：**http://localhost:8000/**
3. 关闭启动时弹出的黑色命令行窗口即可停止服务

### 端口被占用时

若本机 8000 端口已被占用，可修改端口：

- 用记事本打开 `scripts\start.bat`，在 `python -m app.main` 前增加一行：
  ```bat
  set PORT=8080
  ```
  即改为使用 8080 端口，浏览器访问 **http://localhost:8080/**

## 三、页面操作流程（最简单）

1. **创建项目**：输入项目名称 → 点击「创建」
2. **选择项目**：点击「刷新项目列表」→ 在下拉框选择刚创建的项目
3. **上传施组**：在「施组文件」选择施组文件 → 点击「上传施组」（仅上传，进入待评分）
4. **评分施组**：点击「评分施组」后才会计算分数并保存
5. **对比与洞察**：上传并评分多份施组后，可点击「对比排名」「对比报告」「洞察」「生成学习画像」
6. **自适应优化**：需要时点击「自适应建议」→「生成补丁」→「验证效果」→「应用补丁」（应用补丁为可选，需配置 API Key）

6. **自我学习与进化**（提升编制得分）
   - **投喂包**：上传或粘贴项目背景文本（招标文件、清单、图纸、设计等合并的 .txt，如「AI投喂包_含房建+景观+市政等_xxx.txt」），保存到当前项目。
   - **真实评标录入**：将交易中心经青天大模型真实评标后的施组全文 + 5 个评委得分 + 最终得分，粘贴并填写后点击「录入一条」。可多次录入多条。
   - **学习进化**：点击「学习进化」，系统会根据已录入的真实评标结果分析高分逻辑并生成编制指导。
   - **编制指导**：点击「查看编制指导」，查看基于真实评标学习得到的编制建议，用于指导后续施组编制以获得更高分。

数据与配置均保存在本机项目目录下的 `data/` 与 `app/resources/` 中，无需额外配置数据库。

## 四、智能与自动化说明

- **手动触发评分**：上传施组后先进入待评分，点击「评分施组」才执行 16 维度打分
- **自动学习**：「生成学习画像」后，后续评分会按项目历史自动微调权重
- **自动建议**：「自适应建议」根据历史扣分给出词库与规则优化建议；「应用补丁」可一键写回配置并自动备份
- **自动打开浏览器**：启动后约 2.5 秒自动打开系统页面，减少手动输入地址
- **自我学习与进化**：录入真实评标结果（5 评委 + 最终得分）后，系统学习高分逻辑并给出编制指导，用于提升后续施组得分

## 五、常见问题

| 现象 | 处理 |
|------|------|
| 双击 start.bat 闪退 | 先在本机安装 Python 并勾选加入 PATH；或在 cmd 中进入项目目录执行 `python -m app.main` 查看报错信息 |
| 提示缺少某模块 | 在项目目录执行 `python -m pip install -r requirements.txt` |
| 浏览器未自动打开 | 手动打开浏览器，访问 http://localhost:8000/（若改过端口则用对应端口） |
| 中文乱码 | 系统已按 UTF-8 输出；若仍有乱码，请检查浏览器编码是否为 UTF-8，或换用 Chrome/Edge |

## 六、可选：公司内统一配置

- 若需限制“创建项目 / 应用补丁”等操作，可在项目根目录复制 `.env.example` 为 `.env`，设置 `API_KEYS=公司约定密钥`，并在需要写操作的页面按提示输入该密钥。
- 每台电脑数据独立；若需集中存储，需另行部署一台共享服务器并修改数据目录配置（超出本文范围）。
